Create API/.env file with database credentials.

```cd API```

```composer install```

```composer update```

```php artisan migrate```

```php artisan db:seed```

```npm install```

```npm run build```

---

```cd WWW```

```npm install```

```npm run build``` or ```npm start```

---

And you are ready to go after editing ```./WWW/src/js/config/index.tsx``` api.baseURL entry.
